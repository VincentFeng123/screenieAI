# Screenie AI

Screenie AI is a Tauri menu-bar app for capturing a screen region and asking a vision-capable AI provider about it.

## Development

- `npm run dev` starts the Vite frontend.
- `npm run tauri dev` starts the desktop app.
- `npm run build` type-checks and builds the frontend.
- From `src-tauri`, `cargo check` and `cargo test` validate the Rust side.
